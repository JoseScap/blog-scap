# Pagina 1

## Introducción

Azure Container Registry (ACR) es un servicio de registro de Docker administrado y privado que se basa en Docker Registry 2.0 de código abierto. Cree y mantenga los registros de Azure Container para almacenar y administrar las imágenes privadas del contenedor Docker.

## Objetivos

- Explicar las características y ventajas que ofrece Azure Container Registry
- Describir cómo se usa ACR Tasks para automatizar compilaciones e implementaciones
- Explicar los elementos de un Dockerfile
- Compilar y ejecutar una imagen en ACR mediante la CLI de Azure
